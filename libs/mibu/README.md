#  mibu

**mibu** is pure Zig library for low-level terminal manipulation.

> Tested with zig version `0.10.0-dev.2978+803376708`

## Features
- Allocation free.
- UTF-8 support.
- Style (bold, italic, underline, etc).
- Raw mode.
- 8-16 colors.
- True Color (24-bit RGB).
- Cursor controls.
- Clear(Erase) functions.
- Key events.
